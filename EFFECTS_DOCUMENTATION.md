## 📋 功能列表

### 1. 🔥 连击火焰特效

**触发条件：** 连击数 ≥ 10

**效果等级：**
| 连击数 | 等级 | 火焰数量 | 颜色 |
|-------|------|---------|------|
| 10-29 | Level 1 | 5 个 | 黄色→红色 |
| 30-49 | Level 2 | 10 个 | 红色→金色 |
| 50+ | Level 3 | 15 个 | 紫色→红色→金色→白色 |

**动画效果：**
- 火焰跳动（flicker）
- 底部光晕脉冲
- 渐变过渡
- 透明度变化

**位置：** 连击数显示下方

---

### 2. ✨ 按键粒子效果

**触发时机：** 每次按键输入

**粒子类型：**

#### 正确输入 💚
```typescript
颜色：['#4ECDC4', '#FFE66D', '#95E1D3', '#43e97b']
数量：8 个
效果：青绿色系，清新活泼
```

#### 错误输入 ❤️
```typescript
颜色：['#FF6B6B', '#FF8B94', '#ff6b6b', '#f12711']
数量：15 个（更多）
效果：红色系，警示明显
```

**物理效果：**
- 初始速度：2-3 随机
- 重力加速度：0.1
- 生命周期：1.0（约 50 帧）
- 自动清理：16ms 间隔

**生成位置：** 输入框中心

---

### 3. 📳 屏幕震动效果

**触发条件：** 输入错误

**震动强度：**
| 类型 | 持续时间 | 幅度 | 使用场景 |
|-----|---------|------|---------|
| Light | 200ms | 小 | 轻微提示 |
| Medium | 300ms | 中 | 错误输入 ✅ |
| Heavy | 500ms | 大 | 重大错误 |

**动画曲线：**
```css
0% → 10%: 左上 (-5px, -3px)
20%: 右下 (5px, 3px)
30% → 90%: 交替震动
100%: 回到中心
```

## ⚡ 性能优化

### 粒子系统
```typescript
// 清理过期粒子（每帧）
particles.value = particles.value.filter(p => {
  p.life -= 0.02  // 生命衰减
  p.x += p.vx     // 位置更新
  p.y += p.vy     // 垂直移动
  p.vy += 0.1     // 重力
  return p.life > 0  // 清理死亡粒子
})
```

### 动画优化
- ✅ 使用 CSS transform（GPU 加速）
- ✅ 避免重排重绘
- ✅ 限制粒子数量（最多 15 个）
- ✅ 自动清理机制

## 🔧 自定义配置

### 火焰特效配置
```typescript
interface FireConfig {
  minCombo: number      // 最小连击（10）
  levels: number        // 等级数（3）
  flameCounts: number[] // 各等级火焰数 [5, 10, 15]
}
```

### 粒子配置
```typescript
interface ParticleConfig {
  correctCount: number   // 正确粒子数（8）
  wrongCount: number     // 错误粒子数（15）
  lifeTime: number       // 生命周期（50 帧）
  gravity: number        // 重力（0.1）
}
```

### 震动配置
```typescript
interface ShakeConfig {
  durations: {           // 持续时间
    light: number        // 200ms
    medium: number       // 300ms
    heavy: number        // 500ms
  }
}
```
