## 🎮 游戏玩法

1. **选择关卡** - 从易到难解锁关卡
2. **输入文字** - 按照屏幕提示准确输入
3. **保持连击** - 连续正确输入会加速
4. **完成挑战** - 打破自己的记录!

### 节奏机制

- ✅ 连续正确 → 速度加快 (最高 +50%)
- ❌ 输入错误 → 速度减慢 (-20%~-40%)
- 🔥 高连击 → 视觉效果更华丽
- 🔊 音效反馈 → 按键音、连击音、完成音

## 🎹 音效说明

### 音效类型

1. **按键音** - 每次正确输入触发
2. **连击音** - 每 5 连击触发，音调递增
3. **错误音** - 输入错误时触发
4. **完成音** - 关卡完成时播放和弦庆祝
5. **背景音乐** - 可选循环 BGM

### 音效设置

所有音效设置保存在 `localStorage`，键名：`vortex_keyboard_sound_settings`

设置项：
- `enabled`: 音效总开关
- `volume`: 音效音量 (0-1)
- `musicEnabled`: 背景音乐开关
- `musicVolume`: 音乐音量 (0-1)

## 🔧 开发笔记

### 速度计算
```typescript
const bonus = Math.min(combo * 0.05, 0.5) // 最高 +50%
speed = BASE_SPEED * (1 + bonus)
```

### 错误惩罚
```typescript
const penalty = combo > 5 ? 0.4 : 0.2 // 高连击惩罚更重
speed = Math.max(BASE_SPEED * (1 - penalty), BASE_SPEED * 0.5)
```

### 数据存储
- 游戏进度：`vortex_keyboard_stats`
- 音效设置：`vortex_keyboard_sound_settings`
